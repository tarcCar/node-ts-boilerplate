# Projeto Boilerplate de node com typescript

Esse projeto é somente a estrutura de pasta e alguns códigos de exemplo

## Principais pacotes utilizados para o projeto

- Invesify: Para injeção de dependencia
- Express: Para controle de rotas
- Express-validator: Para fazer validação dos End points
- Inversify-express-utils: Framework que tem varios códigos prontos para utilizar express com inversify de um maneira mais facíl
- typeorm: ORM para fazer a integração com o banco de dados
